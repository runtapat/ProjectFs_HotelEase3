🏨 HotelEase - ระบบบริหารจัดการและจองห้องพักโรงแรม

📝 คำอธิบายโครงงาน
HotelEase เป็นเว็บแอปพลิเคชันสำหรับบริหารจัดการโรงแรมและจองห้องพัก พัฒนาขึ้นด้วยสถาปัตยกรรมแบบแยกส่วนระหว่าง Front-End และ Back-End โดยใช้ Node.js และ Express เป็นเฟรมเวิร์กหลักในการประมวลผล และใช้ SQLite ร่วมกับ Sequelize ORM ในการจัดการฐานข้อมูล ระบบนี้ถูกออกแบบมาเพื่ออำนวยความสะดวกให้กับทั้งพนักงานในการบริหารจัดการข้อมูล และลูกค้าในการค้นหาที่พักที่ว่างจริงตามช่วงเวลาที่ต้องการ

## 🚀 รายการฟีเจอร์ที่ทำ
### 🧑‍💻 สำหรับลูกค้า (Customer)
* **ระบบค้นหาและกรองข้อมูล:** ค้นหาโรงแรมตามตำแหน่งที่ตั้ง งบประมาณ และช่วงวันที่เข้าพัก
* **ระบบตรวจสอบห้องว่าง (Availability Check):** คำนวณและแสดงผลเฉพาะห้องพักที่ว่างจริงในช่วงเวลาที่ระบุ เพื่อป้องกันการจองซ้ำซ้อน
* **ระบบคำนวณราคาอัตโนมัติ:** สรุปยอดรวมค่าที่พักตามจำนวนคืนก่อนยืนยันการจอง
* **ประวัติการจองส่วนตัว:** เรียกดูรายการจองย้อนหลัง พร้อมสถานะปัจจุบัน (รอดำเนินการ, ยืนยันแล้ว, ยกเลิกแล้ว)
* **การขอยกเลิกการจอง:** ลูกค้าสามารถยกเลิกรายการจองที่ยังมีสถานะเป็น "รอดำเนินการ" (Pending) ได้เอง

### 👨‍💼 สำหรับพนักงาน (Admin)
* **แดชบอร์ดสรุปสถิติ (Analytics Dashboard):** แสดงรายได้รวมที่ยืนยันแล้ว และยอดเงินที่รอดำเนินการแยกตามรายโรงแรม
* **รายงานสมาชิกเชิงลึก:** ตรวจสอบประวัติการจองและยอดค้างชำระแยกตามรายชื่อสมาชิก
* **การจัดการข้อมูลหลัก (CRUD):** เพิ่ม แก้ไข และลบ ข้อมูลโรงแรมและห้องพัก พร้อมระบบตรวจสอบความสัมพันธ์ของข้อมูลก่อนการลบ
* **การจัดการธุรกรรม:** ตรวจสอบและอัปเดตสถานะการจองของลูกค้าเพื่อยืนยันการเข้าพัก.

### 🔒 ระบบความปลอดภัยและเทคนิค
* **การยืนยันตัวตน:** ระบบสมาชิกใช้ JSON Web Token (JWT) และการเข้ารหัสรหัสผ่านด้วย bcryptjs
* **ระบบคัดกรองสิทธิ์ (Middleware):** ป้องกันการเข้าถึงส่วนงานหลังบ้านหากไม่มีบทบาทเป็นพนักงาน
* **Data Seeding:** มีระบบเติมข้อมูลโรงแรมและห้องพักตัวอย่างเพื่อความรวดเร็วในการทดสอบระบบ
